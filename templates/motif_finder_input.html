Motif finder

This program can scan a set of sequences for matches to each of the motifs provided.

Input your motif(s) (For multiple motifs separate them by new lines or semicolons ';')


Or upload a file with motifs in a column (Allowed file format: all file formats)





Upload a fasta file with sequences for scanning(Allowed file format: fasta file - fasta\fna\fa\faa)



Tips: Both of uppercase or lowercase nucleotide in the input text\file are allowed.



If you need change the input values, click the "Reset" button, the form-data will be reset to the blank.

Tips: You can download the produced table and fasta files that contain only the sequences that have the motif.
